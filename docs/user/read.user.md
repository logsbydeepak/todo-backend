# Read User

Get user information

- **URL**

`GET /v1/user`

`AUTHENTICATION`

- **URL Params**

  None

- **Data Params**

None

- **Success Response:**

  - **Code:** 200

    **Content:**

    ```json
    {
      "data": {
        "name": "test user",
        "email": "test@email.com"
      }
    }
    ```
